Lección 1: Variables y tipos de datos

En Python, no necesitas declarar el tipo de variable. Aquí un ejemplo:

x = 10
nombre = "Ana"
es_estudiante = True
print(x, nombre, es_estudiante)

Salida:

10 Ana True

Lección 2: Condicionales

La estructura condicional permite tomar decisiones:

edad = 18
if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")

Salida:

Eres mayor de edad

Lección 3: Bucles

Python tiene bucles for y while:

# Bucle for
for i in range(5):
    print(i)

# Bucle while
contador = 0
while contador < 5:
    print(contador)
    contador += 1

Lección 4: Funciones

Las funciones son bloques de código reutilizables:

def saludar(nombre):
    print(f"Hola, {nombre}!")

saludar("Carlos")

Salida:

Hola, Carlos!
